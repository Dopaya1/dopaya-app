# Brand logos

Place brand logo images here so they appear on the homepage when not provided by Supabase.

**Expected filenames** (from `src/data/brands.json`):

- `nikin-logo.png`
- `rrrevolve-logo.png`
- `2nd-peak-logo.png`
- `reslides-logo.png`

Recommended: PNG or WebP, transparent background, max height ~128px.  
If logos are set in Supabase (`brands.logo_url`), those URLs are used instead.
